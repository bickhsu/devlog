# ContextBadge

DevLog's signature context-path pill. Renders /a/b/c with faint slashes and an emphasized leaf. Click to open the column picker.

```jsx
<ContextBadge path="/metai/cad-agent/adapter" onClick={openPicker} active={pickerOpen} />
<ContextBadge path="/metai/hython-server" size="sm" interactive={false} />
```
